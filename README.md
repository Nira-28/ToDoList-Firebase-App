📝 ToDoList – Android App with Firebase Integration
📋 Overview

ToDoList is a simple and elegant Android app built using Android Studio and integrated with Firebase for real-time data storage and authentication.
This app allows users to add, update, and delete tasks — helping them organize their daily goals efficiently.

🚀 Features

✅ User authentication using Firebase Authentication (Email/Password)
✅ Real-time task storage using Firebase Firestore
✅ Add, edit, and delete tasks easily
✅ Intuitive and responsive UI built with Material Design
✅ Secure login and logout functionality
✅ Cloud synchronization — your tasks are available on any device

🧠 Tech Stack

Language: Java / Kotlin

IDE: Android Studio

Database: Firebase Firestore

Authentication: Firebase Auth

Design: XML (Material Design Components)

⚙️ Firebase Configuration

This project uses Firebase, so the google-services.json file is not included in this repository for security reasons.
Each developer must create their own Firebase configuration.

🔧 To set up Firebase:

Go to Firebase Console
.

Create a new Firebase project.

Add your Android app (use your app’s package name).

Download the google-services.json file.

Place it in your project at:

app/google-services.json


Sync your project with Gradle files.
